# Work_Day_Scheduler

This is a simple calendar app which allows a user to save activities or events for the workday hours.  This app will run in the browser utilizing the computer local storage and will dynamically update the HTLM and CSS powered by JQuery.  

## User Story
AS A parent with a busy schedule
I WANT to add important events to a daily planner
SO THAT I can manage my time and responsibilities effectively

## How to use
Each hour has a text area for the user to input the event or activity
Each hour has a save button
The save button will store the user input to the computer local storage
The local storage will also load when calendar is opened
